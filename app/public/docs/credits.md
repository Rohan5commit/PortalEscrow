# Credits

PortalEscrow uses and credits these open-source tools and ecosystems:

- Rust and Cargo for contract development.
- ink! smart contract framework for Substrate-compatible contracts.
- SCALE codec and scale-info for contract data encoding and metadata.
- Substrate/Polkadot.js APIs for wallet, RPC, and contract interaction.
- Substrate browser wallet extension ecosystem for account signing.
- Next.js and React for the frontend application.
- TypeScript for typed frontend logic.
- Tailwind CSS, PostCSS, and Autoprefixer for styling.
- Vercel for frontend hosting.
- Portaldot ecosystem documentation and network tooling for deployment guidance.
