# Judging Hook

PortalEscrow is built to score well because it is immediately understandable and technically real.

## Native deployment

The product is designed for Portaldot from the contract layer up: an ink! contract, Substrate wallet flow, Portaldot RPC configuration, and POT gas in wallet-confirmed transactions.

## Clear demo

The demo is a simple story: client creates and funds a job, freelancer submits work, client releases milestone 1, and freelancer claims milestone 2 after timeout.

## Real application value

Freelance payment risk is easy to understand. Clients want controlled release. Freelancers want assurance that funds are locked before work begins. PortalEscrow addresses both sides with transparent rules.

## Strong presentation

The app includes a landing page, dashboard, escrow detail page, event summaries, architecture page, setup guide, demo script, and Devpost-ready copy.
